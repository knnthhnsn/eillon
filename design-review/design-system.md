# EILLON Unified Design System

**Date:** 2026-06-18  
**Source of truth:** styles.css :root tokens

## Color tokens

- --color-canvas (#FAFAF8) page background
- --color-paper (#FFFFFF) cards
- --color-stone-light (#ECECEA) panels
- --color-ink / --color-ink-soft / --color-muted text hierarchy
- --color-line borders
- --color-sand, --color-blush, --color-sea, --color-resin scent accents (subtle only)

Legacy aliases (--bg, --ink, --gold) remain.

## Typography scale

- --text-display-xl: homepage hero
- --text-display-lg: journal, section heads
- --text-heading-lg/md/sm: chapters, cards
- --text-body-lg/body/body-sm: copy
- --text-label / --text-caption: UI chrome

Families: Fraunces (serif), Inter (sans). Aliases: --font-serif, --font-sans.

## Spacing

--space-1 through --space-8 (8px-88px), --space-section for vertical rhythm.

## Layout

--page-max 1480px, --content-max 720px, --text-max 58ch, --page-gutter, --grid-gap.

## Motion

--ease, --ease-out, --duration-reveal (0.85s), --duration-fast (0.35s), --duration-editorial (1.4s).

## Component rules

- Navigation: light, precise; quiet Restock pill
- Buttons: primary ink, ghost secondary; pills for CTAs only
- Boutique cards: image-first, caption overlay, Beles lead row, Ritual archival
- Forms: underline inputs, serif placeholders, calm focus
- Palette: chapter index eyebrows, border-led cards
